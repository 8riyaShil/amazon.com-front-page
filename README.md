# 🛒 Amazon Clone

A fully responsive **Amazon Homepage Clone** built using **HTML5** and **CSS3**. This project recreates the look and feel of Amazon's landing page, including the navigation bar, hero section, product categories, horizontal product sliders, and footer sections.

## 🌐 Live Demo

🔗 https://meek-cassata-564bbd.netlify.app/

---

## ✨ Features

* 🏠 Amazon-style Homepage
* 🔍 Search Bar UI
* 📍 Delivery Location Section
* 🌎 Language Selector
* 👤 Sign In & Orders Section
* 🛒 Shopping Cart UI
* 📋 Navigation Panel
* 🎮 Product Category Cards
* 🖼️ Hero Banner Section
* 📚 Horizontal Product Sliders
* 📱 Responsive Layout
* ⬆️ Back to Top Button
* 🦶 Amazon-style Footer
* 🎨 Clean and Modern Design

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/8riyashil/amazon-clone.git
```

### Open the Project

```bash
cd amazon-clone
```

Open `index.html` in your browser.
